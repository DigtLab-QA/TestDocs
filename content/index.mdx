---
title: "Руководство пользователя Trusted.Net"
---

Trusted.Net — сервис, позволяющий использовать учетные записи пользователей из различных типов хранилищ для авторизации в сторонних информационных системах c помощью технологии single sign-on.  

Предоставляет следующие возможности:  
- управление учетными данными пользователя– создание, редактирование, блокировка, удаление;   
- управление входом и правами доступа в информационную систему через настройки приложения;  
- регулирование пользователем разрешений, которые он предоставляет информационной системе.  

Trusted.Net позволяет использовать следующие способы аутентификации:  
 - простая аутентификация (логин+пароль);  
 - аутентификация через социальные сети;  
 - атуентификация по номеру телефона и e-mail.  


Trusted.Net поддерживает возможность интеграции с доверенными источниками для загрузки или обновления данных о пользователях.  

Поддерживаемые доверенные источники:  
 - LDAP — каталоги и доменные сервисы;  
 - веб-сервисы по протоколу OAuth 2.0.

Доступны интеграционные модули с приложениями:  
- 1С-Битрикс и Битрикс24 (коробочное решение);  
- КриптоАРМ Документы - модуль для подписи электронных документов для 1С-Битрикс.
